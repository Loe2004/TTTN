Các bước run project

- tạo folder rỗng tên j cũng dc
- Xong mở terminal trong folder đó
- chạy "git clone https://github.com/Loe2004/TTTN.git"
- xong chạy lệnh tạo máy ảo "python -m venv venv"
- xong khởi tạo máy ảo "venv\Scripts\activate"
- xong download file requiremetn về lệnh là: "pip install -r requirements.txt"
- xong chạy thôi "python manage.py runserver" là có đường dẫn dến web
